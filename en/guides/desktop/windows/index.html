Desktop workflow guide

This guide covers the shared AiyoPerps desktop workflow for Windows and Linux, from account setup to Dashboard usage and exchange-specific workspace tabs.

Best for

Users who already installed the Windows or Linux desktop app.

Estimated time

About 10 to 15 minutes.

Outcome

A working workspace ready for manual trading or AI-agent collaboration.

On this page

Account ManagerDashboard workflowExchange tab workflowNext step

1Use Account Manager

Open the account management window and add or edit exchange credentials. You need at least one configured exchange account before you can trade. For CEX venues this usually means API key and API secret. For DEX venues this may require your public wallet address, an exchange-generated API wallet address, and an API wallet key. For security, do not enable withdraw permission on any exchange API credential used with this app.

2Work from the Dashboard

The Dashboard is the default startup view. The right side manages multiple accounts across venues, while the left side is reserved for AI agent integration settings. Enable the accounts you want to trade from this screen, choose the contract symbol in the unified order entry panel, then use the live market area, positions area, and orders area for day-to-day operation.

3Use exchange-specific tabs

Besides the Dashboard, AiyoPerps also offers dedicated exchange workspace tabs. These tabs provide a more detailed live trading view, including charting and exchange-specific context. Click + New Tab, choose the exchange account for the new tab, confirm it, then select the symbol and candle interval once the tab is active.
