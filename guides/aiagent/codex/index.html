AI Agent 安裝及互動指引

AiyoPerps 可以設定多種條件主動喚醒 AI Agent 介入操作，也可以被動接受 AI Agent 主導操作，本篇以 OpenAI Codex 為例說明安裝及設定流程。

適用對象

想要透過 AI Agent 查詢或操作永續合約交易的使用者

預估時間

約 10 到 15 分鐘

完成後你會得到

AiyoPerps 能夠與 AI Agent 循環串聯協作

本頁內容

安裝 OpenAI Codex CLI 安裝 AiyoPerps MCP 設定 AI Agent 下一步

1安裝 OpenAI Codex CLI

  • 若系統已安裝 Codex CLI 請略過本步驟
  • Codex App 也可以透過 MCP 操作 AiyoPerps，但無法接受 AiyoPerps 以條件式喚醒，所以建議以 Codex CLI 為主
  • OpenAI 官方建議 Windows 的使用者將 Codex CLI 安裝在 WSL 內會較為順暢。安裝方式請參考 OpenAI 官方網站
  • 執行 Codex CLI，確保可以正常運作

2安裝 AiyoPerps MCP

  • 安裝 AiyoPerps MCP 可以加速 AI Agent 操作需要的時間，節省 token 消耗
  • AiyoPerps MCP 只允許本機連接 (包含 WSL 虛擬網路介面)
  • Linux 及 Windows 皆可使用以下命令安裝 AiyoPerps MCP
npx -y @phidiassj/aiyoperps-mcp-installer

3設定 AI Agent

  • 點擊 Dashboard 左上方的「代理設定」即可開啟 AI Agent 設定視窗
  • 開啟「啟用 AI Agent」讓本介面設定生效
  • 「喚醒週期」可設定以分鐘為單位喚醒 AI Agent，設定為 0 關閉定時喚醒
  • 「Prompt 樣板」設定您的提示詞，可以是操作策略、資料分析等，可以帶入 MCP 及 Rest API 工具手冊 中的命令及參數
  • 「條件喚醒」可設定以價位或持倉獲利率為條件喚醒 AI Agent
