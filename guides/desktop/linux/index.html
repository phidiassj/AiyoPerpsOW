Linux 桌面版操作教學

這篇頁面延續 Linux 安裝流程，帶你完成帳號設定、工作區建立、商品選擇與面板操作。

適用對象

已安裝 Linux 桌面版的使用者

預估時間

約 10 到 15 分鐘

完成後你會得到

一個可供桌面操作與 API 整合的工作區

本頁內容

步驟 1 帳號管理步驟 2 啟用工作區步驟 3 設定商品步驟 4 面板操作下一步
1

新增或編輯交易帳號

aiyoperps-desktop-linux-step-01-open-account-manager.png

打開 Account Manager，新增對應交易所帳號，確認之後要綁定到 tab 的資料已經準備完成。

建議放置：Linux 桌面上的 Account Manager 視窗。

2

建立 tab 並啟用帳號

aiyoperps-desktop-linux-step-02-create-workspace-tab.png

新增 tab、選擇帳號與平台後確認啟用。這會建立 Linux 端的實際工作區。

建議放置：Linux 下新增 tab 與選擇帳號的畫面。

3

選擇 Symbol 與市場資料週期

aiyoperps-desktop-linux-step-03-set-symbol-and-interval.png

完成 tab 啟用後，設定 Symbol 與 Interval，讓圖表與工作區進入你要監看的市場狀態。

建議放置：Linux 工作區中的 Symbol 與 Interval 設定區塊。

4

熟悉右側操作面板

aiyoperps-desktop-linux-step-04-open-order-panel.png

先從 `Order` 面板開始熟悉，再切換 `Positions`、`Orders`、`Balances` 等頁面理解整體操作流。

建議放置：Linux 桌面版右側面板切換的實際畫面。
