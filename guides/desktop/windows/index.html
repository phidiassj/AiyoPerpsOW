Windows 桌面版操作教學

安裝完成後，這篇教學會帶你完成帳號建立、開啟 tab、設定商品與進入實際操作工作區。

適用對象

已安裝 Windows 桌面版的使用者

預估時間

約 10 到 15 分鐘

完成後你會得到

一個可以開始交易或連接 AI Agent 的工作區

本頁內容

步驟 1 帳號管理步驟 2 新增 tab步驟 3 設定商品步驟 4 開始操作下一步
1

打開 Account Manager 新增帳號

aiyoperps-desktop-windows-step-01-open-account-manager.png

先進入帳號管理畫面，新增或編輯交易所帳號，確保之後建立工作區時有可用帳戶。

建議放置：Account Manager 入口與帳號清單畫面。

2

新增一個 tab 並綁定帳號

aiyoperps-desktop-windows-step-02-create-workspace-tab.png

按下 `+` 新增 tab，選擇平台與帳號後確認啟用。這個 tab 會成為你的實際交易工作區。

建議放置：新增 tab 與選擇帳號的流程畫面。

3

設定 Symbol 與 Interval

aiyoperps-desktop-windows-step-03-set-symbol-and-interval.png

進入工作區後，先選擇商品與 K 線週期。這會影響圖表、委託簿與後續的下單情境。

建議放置：工具列中 Symbol 與 Interval 的設定畫面。

4

使用右側面板進行操作

aiyoperps-desktop-windows-step-04-open-order-panel.png

右側面板可切換 `Order`、`Positions`、`Orders`、`Balances` 等分頁。先從下單區塊熟悉介面，再逐步切換其他視圖。

建議放置：Order 面板與 Positions / Orders / Balances 切換狀態。
