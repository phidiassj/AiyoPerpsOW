桌面版操作教學

包含 Windows 及 Linux 桌面版，這篇教學會帶你完成帳號建立、開啟 tab、設定商品與進入實際操作工作區。

適用對象

已安裝 Windows 或 Linux 桌面版的使用者

預估時間

約 10 到 15 分鐘

完成後你會得到

一個可以開始交易或連接 AI Agent 的工作區

本頁內容

帳號管理 使用 Dashboard 使用交易所獨立 Tab 頁籤 下一步

1Account Manager 帳號管理

先開啟帳號管理視窗，新增或編輯交易所憑證，您必須有任何一個交易所憑證才能開始交易。
CEX 中心化交易所憑證通常是 API Key / Id (交易所生成)及 API Secret (交易所生成)。
DEX 去中心化交易所憑證通常需要 您的錢包公開位址、API 錢包位址 (交易所生成)、API 錢包 Key (交易所生成)。
安全起見，無論是 CEX 或 DEX，在交易所建立 API 憑證時，請務必不要開啟提領(Withdraw)權限，您不需要提領權限即可使用本系統完整功能。

2使用 Dashboard

本系統啟動後的畫面預設就是 Dashboard，右方提供跨平台多帳號同步管理功能，左邊是 AI Agent 整合設定區。
1. 在啟用帳號勾選要在本介面操作的交易所帳號。
2. 在合約選擇整合下單面板的操作幣別。
3. 啟動後，即時資訊區、持倉管理、訂單管理 即可開始操作。
AI Agent 相關資訊請參考 「AI Agent 安裝及互動指引」 章節

3使用交易所獨立 Tab 頁籤

除了 Dashboard 介面，本系統同時提供獨立的交易所操作介面，相比 Dashboard，獨立頁籤提供更完整的即時資訊，包括 K線圖(蠟燭圖) 等。
1. 按一下主視窗左上角的 + 新增TAB。
2. 在新增的頁籤，下拉選單選擇交易所帳號，按確定。
3. 啟動後可再選擇合約(幣別)、K線圖時間單位等。
