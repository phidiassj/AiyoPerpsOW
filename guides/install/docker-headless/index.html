如何用 Docker 啟動 AiyoPerps Headless

如果你只需要本機 REST API 與 MCP，而不需要桌面 GUI，這篇頁面會帶你直接用 Docker 啟動 AiyoPerps 的 headless 模式。

適用對象

Mac 使用者
只使用 AI Agent 操作 (無 GUI 介面)
只需要 API / MCP 與自動化節點

預估時間

約 5 分鐘

完成後你會得到

對外提供 `5078` 埠號的 headless 容器

本頁內容

前置需求 步驟 1 拉起容器 步驟 2 檢查埠號 步驟 3 驗證 API 下一步
Requirements

前置需求

  • 已安裝 Docker
  • 可開放本機 `5078` 埠號
  • 需要 REST API 或 MCP 連接能力

1使用 docker run 啟動容器

官方 README 目前提供的方式是直接啟動最新映像並將本機 `5078` 映射進容器。

docker run --rm --name aiyoperps -p 5078:5078 phidiassj/aiyoperps:latest

2確認容器與埠號狀態

確認容器正在執行，並且 `5078` 已經正確綁定到本機。

docker ps

3驗證 OpenAPI 或 MCP 端點

在瀏覽器開啟 OpenAPI UI，或直接對 MCP 端點做連線測試。只要頁面能打開，就表示 headless 服務正常。

http://127.0.0.1:5078/scalar http://127.0.0.1:5078/mcp
