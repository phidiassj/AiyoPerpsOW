Install guide

如何在 Linux x64 安裝 AiyoPerps

這篇頁面整理 Linux x64 版本的下載、解壓、授權與啟動流程，適合 Linux 桌面或工作站使用者。

適用對象

Linux x64 使用者

預估時間

約 10 分鐘

完成後你會得到

可在 Linux 直接啟動的桌面版環境

本頁內容

前置需求 步驟 1 下載步驟 2 解壓與權限 步驟 3 啟動驗證
Requirements

前置需求

  • Linux x64 桌面環境
  • 可使用終端機(命令列)
  • 有權限修改檔案執行權限

1使用 AppImage 安裝 (推薦)

aiyoperps-install-linux-x64-step-01-download-release.png

在有寫入及執行權限的路徑開啟終端機，使用如下命令安裝最新版本並執行。

wget https://github.com/phidiassj/AiyoPerps/releases/latest/download/AiyoPerps-x86_64.AppImage chmod +x AiyoPerps-2026.04.03-x86_64.AppImage ./AiyoPerps-2026.04.03-x86_64.AppImage
2

解壓並設定執行權限

aiyoperps-install-linux-x64-step-02-extract-and-chmod.png

把壓縮檔解開後，進入資料夾並為主程式加上執行權限。

tar -xf aiyoperps-linux-x64.tar.gz cd AiyoPerps chmod +x AiyoPerps

建議放置：終端機完成解壓與 `chmod +x` 後的畫面。

3

啟動程式

aiyoperps-install-linux-x64-step-03-launch-app.png

在資料夾中執行主程式，確認桌面版視窗能正常開啟。

./AiyoPerps

建議放置：Linux 桌面上成功啟動 AiyoPerps 的主視窗。

Verify

如何確認安裝成功

  • 終端機沒有權限錯誤
  • 桌面主視窗正常顯示
  • 程式關閉前不會立即崩潰
