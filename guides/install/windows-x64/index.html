如何在 Windows x64 安裝 AiyoPerps

這篇教學針對一般桌面使用者，帶你從 GitHub Releases 下載 Windows x64 版本，完成解壓、啟動與基本驗證。

適用對象

第一次安裝 AiyoPerps 的 Windows 桌面使用者

預估時間

約 5 到 10 分鐘

完成後你會得到

可正常啟動的 AiyoPerps 桌面版環境

Requirements

前置需求

  • Windows x64 環境
  • 可存取 GitHub Releases
  • 有權限解壓縮到本機資料夾

建議把程式解壓在你容易找到的位置，例如 `C:\Apps\AiyoPerps` 或使用者目錄下的工具資料夾。

1下載 Windows x64 發行版

前往 GitHub Releases，找到對應的 Windows x64 發行壓縮檔並下載到本機。

2解壓到固定資料夾

把下載的壓縮檔完整解壓，確認資料夾內包含主程式與相依檔案，不要只解出單一執行檔，也不要直接在壓縮檔內執行。

3啟動 AiyoPerps.exe

在解壓後的資料夾中執行 `AiyoPerps.exe`。第一次啟動時，如果系統安全提示出現，請確認來源後允許執行。

AiyoPerps.exe
Verify

如何確認安裝成功

  • 主視窗可以正常打開
  • 沒有立即崩潰或缺檔錯誤
  • 可看到主要工作區與上方工具列
